# SUSKE EN WISKE EN DE DE ALWETENDE API

In this project you can find multiple ways to get access to all that you ever wanted to know about the albums of the famous Suske en Wiske.

## GRAPHQL

### ENV variables

TRPCSERVER=http://localhost:4000/trpc //for production, this is of course the production version
HYVETOKEN=XXXX //for analytics
HYVETARGET=straatemans/suskeenwiske/development // `straatemans/suskeenwiske/production`
SENTRYDSN=XXX
ENV=development // or `production`. used for sentry environment
